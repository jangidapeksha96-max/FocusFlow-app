Built something simple but actually useful this week — a productivity web app called **FocusFlow**.

The idea was not to overcomplicate things, but to create a clean tool that I’d personally use while studying.

🔹 What it does:
• Task manager to keep things organized
• Pomodoro timer with focus + break cycles
• Daily motivational quotes (fetched from API)
• Productivity stats (sessions, focus time, tasks done)
• Dark / Light mode for better experience

🔹 Tech used:
Just **HTML, CSS, and JavaScript** — no frameworks.

I wanted to strengthen my fundamentals instead of jumping straight into libraries. Handling things like:

* localStorage for saving data
* timer logic
* API fetching
* UI interactions

gave me a much better understanding of how things actually work behind the scenes.

Also focused a lot on making the UI feel clean and app-like rather than just functional.

Still improving it step by step (thinking of adding better stats and a more refined UI next).
